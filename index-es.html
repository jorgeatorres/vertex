Acerca de verTeX

verTeX es un aplicativo web especialmente preparado para la edición colaborativa de textos científicos en los que se requieren símbolos matemáticos. Está basado en el sistema LaTeX y por tanto proporciona la mejor calidad en materia tipográfica.

Características

  • Los usuarios de verTeX sólo necesitan una conexión a internet en su propio computador.
  • verTeX está orientado al trabajo colaborativo: puede crear diferentes proyectos de edición, gestionar los permisos de sus colaboradores, invitar nuevos editores.
  • Con el control de versiones incorporado en verTeX no perderá nunca el trabajo realizado: podrá comparar diferentes revisiones de sus documentos, compilar cualquier versión anterior de los mismos o revertir cambios con facilidad.
  • Dentro otras de las cualidades de verTeX se destaca el editor LaTeX incluído, la posibilidad de exportar sus documentos en diferentes formatos (PDF, DVI, PS) o la gestión de versiones de archivos binarios.

Pantallazos

Pantallazos de la versión en desarrollo de verTeX.
(haga click sobre las miniaturas para ver la imagen completa).

  • Vista del listado de archivos de un proyecto y el editor de LaTeX con coloreado de sintaxis.
  • Menú para inclusión de símbolos LaTeX predefinidos.
  • Resultados de una compilación exitosa de un documento, con posibilidad de descargar el resultado en formato PDF, DVI o PS.
  • verTeX gestiona archivos de contenido binario (como imágenes) de manera transparente.
  • Comparación de dos revisiones de un archivo con las diferencias indicadas.

Desarrollo de verTeX

verTeX está desarrollado en Python utilizando el framework TurboGears 2.0 y la librería jQuery del lado del cliente.
Para la compilación de los documentos LaTeX se utiliza el comando latex del sistema y dvipng para generar imágenes PNG de los símbolos incluídos en el editor.

Código fuente

verTeX se encuentra disponible en un repositorio Git en GitHub.

Puede descargar una copia del código fuente desde el repositorio con el siguiente comando: 

$ git clone git://github.com/jorgeatorres/vertex.git
